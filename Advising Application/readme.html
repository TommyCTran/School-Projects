Advising Time Selection Readme

Student Create Group Appointment:
You are able to sign up for any available group appointments beginning 2 business days from today and ending 9 business days from today. If you already have an upcoming appointment, then you will not be able to create a group appointment. If you had a past appointment, then you may not choose this option.

Student Create Individual Appointment:
You are able to sign up for any available individual appointments from an advisor of your choosing beginning 2 business days from today and ending 9 business days from today. If you already have an upcoming appointment, then you will not be able to create a individual appointment. If you had an individual appointment this week, then you will not be able to sign up for one until Monday of next week.

Student Change Group Appointment:
If you have an upcoming appointment, but decide that you need to reschedule, then you may do so. When you pick this option, you can choose an available group appointment. You can not choose this option if your upcoming appointment is within 24 hours. If you had a past appointment, then you may not choose this option. See Student Create Group Appointments for more information.

Student Change Individual Appointment:
If you have an upcoming appointment, but decide that you need to reschedule, then you may do so. When you pick this option, you can choose an available individual appointment for an advisor of your choosing. You can not choose this option if your upcoming appointment is within 24 hours. See "Student Create Group Appointments" for more information.

Student View Appointment Details:
This page will show, in detail, your past appointments and you upcoming appointments. If you don't have any made appointments, then you cannot view this page.

Student Cancel Upcoming Appointment:
If you have an upcoming appointment that you wish to cancel, you can choose this option and confirm it on the next page. If your upcoming appointment is within 24 hours, then you may not cancel your appointment. You must give at least 24 hours before the appointment to cancel it. If you don't have any upcoming appointments, then you can not use this option.

Advisor View Schedule:
Choose this option to view the schedule of any advisor and every day within the time frame of course registration approval. The default advisor is the current user and the defualt day is today, both of which can be chosen in the drop-down menus at the top of the page. Depending on the appointment type, the major will either be the student's major (for individual appointments), or the major of the group session, which are one or all of the COEIT majors.

Advisor Create Appointment times:
This is how the advisor chooses times for their appointment. To select a time, check a desired time, and select either group or individual advising for the time slot. If group, check select the number of students in the appointment as well as the major. Options include Computer Science, Computer Engineering, Chemical Engineering, Mechanical engineering, and All Majors. Click 'Next' to make the selected times available. If the user comes into this page again, the times that they previously selected will not be available to select again, unless they cancel the time.

Advisor Cancel Appointment times:
The user may cancel appointment times that they previously made available by selecting this option. When in the Advisor Cancel Appointments page, times show up that were selected previously. The user may check the times they wish to delete. If a student has created an appointment for a certain time slot, a red message that says "Student assigned to time slot" appears next to the time slot. Click 'Next' to cancel the appointment times. If the time was already chosen by a student, both the appointment and the available time are canceled. If the user only wants to cancel an appointment but not the time availability, use the 'Cancel Appointment' option instead.

Advisor Search for Student:
You can type in a student's last name and see all of the information of every student with that last name and all of their past and upcoming appointments, along with the advisor that the appointment was with. You could also type in the Student's ID and minimize your search results. If the student is not in the database, then an error will occur notifying you that the last name or ID is unknown.

Advisor Cancel Appointment:
You can cancel an upcoming appointment for a student at any time as long as you have their Student ID. If you do not know the student's ID, see "Advisor Search for Student" and "Advisor View Schedule" for more help. If the student doesn't have any upcoming appointments, then an error will occur telling you so. If the ID is not in the database, then an error will occur telling you so.

Advisor Reschedule Appointment:
With this option, you can type in a student's ID and reschedule their upcoming appointment with any advisor (even Group Advising). You can reschedule an appointment to today, unlike where students can only reschedule 2 days away. You can reschedule a student to a group appointment even if they had a past appointment. If you do not know the student's ID, see "Advisor Search for Student" and "Advisor View Schedule" for more help. If the student doesn't have any upcoming appointments, then an error will occur telling you so. If the ID is not in the database, then an error will occur telling you so.

Advisor Schedule Appointment for New or Existing Student:
You can schedule an appointment and choose the advisor (or Group Advising) for a new or existing student. If the student is not in the database, then you must fill in all of the new student's information. If the student is existing in the database, then all you have to do is input their Student ID. If the ID is not found, then an error will occur telling you so. If an existing student already has an upcoming appointment, then an error will occur telling you so. If you have information in both the new student side and the existing student side, then you will be told to only fill in one of the two.
